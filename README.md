# CMOS Inverter Design using SKY130 Open PDK

## Overview
- CMOS inverter designed using SKY130 Open PDK
- Schematic capture and pre-layout simulation performed in xschem using ngspice
- Physical layout implemented in Magic VLSI with parasitic extraction
- Post-layout simulation performed using ngspice

## Tools Used
- xschem
- ngspice
- Magic VLSI
- SKY130 Open PDK
